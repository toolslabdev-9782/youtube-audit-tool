# YouTube Channel Audit Tool (v1)

A simple YouTube channel audit tool that analyzes public channel data
and provides mentor-style insights for creators.

## Features (v1)
- Fetches public YouTube channel data
- Provides 3 audit insights:
  1. Content Volume Health
  2. Upload Consistency
  3. Subscriber Conversion Efficiency
- FastAPI backend
- Simple HTML frontend
- Uses official YouTube Data API (public data only)

## Tech Stack
- Python 3.12
- FastAPI
- Uvicorn
- Requests
- HTML + JavaScript (frontend)
